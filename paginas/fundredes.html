O que são redes de computadores?

Redes de computadores conectam dispositivos para compartilhar recursos e informações. Elas permitem que computadores, impressoras, servidores e outros equipamentos se comuniquem.

Rede mínima

Uma rede mínima precisa de pelo menos dois dispositivos conectados. Por exemplo, dois computadores ligados por um cabo Ethernet ou via Wi-Fi.

O que é a internet?

A internet é uma rede global que conecta milhões de redes menores. Inclui redes privadas, públicas, acadêmicas, empresariais e governamentais.

O que compõe a internet?

  • Provedores de internet
  • Roteadores e switches
  • Cabos de fibra óptica, cobre e wireless
  • Servidores e data centers
  • Protocolos de comunicação

O que é a web?

A web é um sistema de documentos e recursos conectados que você acessa pela internet. Usa HTTP para enviar dados e HTML para organizar as páginas.

Mapa de fibra óptica

O mapa global mostra os cabos submarinos e terrestres que conectam continentes. Esses cabos enviam dados rapidamente usando luz.

Por exemplo, cabos como o MAREA ligam Europa à América do Norte pelo Atlântico.

Componentes de hardware de uma rede Wi-Fi

  • Roteador Wi-Fi: conecta sua rede local à internet e distribui o sinal sem fio
  • Pontos de acesso: aumentam o alcance do sinal Wi-Fi
  • Adaptadores Wi-Fi: placas ou dispositivos que permitem conectar computadores à rede sem fio
  • Antenas: melhoram o envio e recebimento do sinal

Componentes de hardware de uma rede local com cabo

  • Switch: conecta vários dispositivos na mesma rede local
  • Cabos Ethernet: cabos de cobre que transmitem dados
  • Placas de rede: interfaces que conectam o computador à rede
  • Roteador: conecta a rede local à internet

Protocolo de comunicação

Um protocolo de comunicação define regras para dispositivos se comunicarem em uma rede. Ele especifica como formatar mensagens, a ordem de envio e como lidar com erros.

Modelo OSI

O Modelo OSI é uma referência para redes de computadores, dividido em 7 camadas:

  1. Física: envia bits
  2. Enlace: controla a conexão e detecta erros
  3. Rede: roteia e endereça com IP
  4. Transporte: controla o fluxo e corrige erros
  5. Sessão: cria e mantém sessões
  6. Apresentação: traduz dados e criptografa
  7. Aplicação: interfaces para programas do usuário

Modelo TCP/IP

O Modelo TCP/IP é usado na internet, com 4 camadas:

  1. Interface de rede: cobre as camadas física e enlace do OSI
  2. Internet: roteia e endereça com IP
  3. Transporte: usa TCP e UDP
  4. Aplicação: protocolos como HTTP, FTP, SMTP

Protocolo IP v4 e v6

IP v4: versão atual, usa endereços de 32 bits (exemplo: 192.168.1.1). Suporta cerca de 4 bilhões de endereços.

IP v6: nova versão, usa endereços de 128 bits (exemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334). Suporta muitos mais endereços e é mais seguro.

Configuração de IP e TCP em Node.js

Em Node.js, para o backend, use o módulo 'net' para TCP e 'http' para HTTP/IP. Configure o host e a porta no servidor.

const net = require('net');

const server = net.createServer((socket) => {
  socket.write('Olá do servidor TCP!');
  socket.end();
});

server.listen(8080, '127.0.0.1', () => {
  console.log('Servidor TCP ouvindo em 127.0.0.1:8080');
});

Protocolo HTTP

HTTP é o protocolo que transfere dados na web. Ele define como formatar e enviar mensagens entre cliente e servidor.

Protocolo HTTPS

HTTPS é a versão segura do HTTP. Usa SSL/TLS para criptografar a comunicação e proteger dados como senhas e informações de cartão.

Framework para HTTP com Node.js

Exemplo: Express.js, um framework para Node.js que facilita criar APIs e aplicações web.

const express = require('express');

const app = express();

app.get('/', (req, res) => {
  res.send('Olá, mundo!');
});

app.listen(3000, () => {
  console.log('Servidor rodando na porta 3000');
});

Exemplos de código backend para troca de mensagens

Exemplo 1: Servidor HTTP simples

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Mensagem do servidor');
});

server.listen(3000);

Exemplo 2: API REST com Express

const express = require('express');

const app = express();

app.use(express.json());

app.post('/message', (req, res) => {
  const message = req.body.message;
  res.json({response: `Recebido: ${message}`});
});

app.listen(3000);

Exemplo 3: WebSocket para mensagens em tempo real

const WebSocket = require('ws');

const wss = new WebSocket.Server({port: 8080});

wss.on('connection', (ws) => {
  ws.on('message', (message) => {
    ws.send(`Eco: ${message}`);
  });
});
