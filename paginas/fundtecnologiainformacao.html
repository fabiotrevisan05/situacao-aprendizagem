Hardware de computadores

Hardware são as partes físicas do computador, como processador, memória, disco rígido e outros.

Placa mãe

É a placa principal que conecta tudo no computador. Serve de base para ligar CPU, RAM, discos e mais.

Processador (CPU)

É o cérebro do computador, roda instruções e processa dados. Exemplos: Intel Core i7, AMD Ryzen.

Memória principal (RAM)

Memória temporária que guarda dados em uso. É volátil e apaga tudo quando o computador desliga.

Sistema operacional

Sistema operacional é o software que controla o hardware e dá interface para o usuário rodar programas.

Exemplos:

  • Windows: da Microsoft, mais comum em desktops
  • Linux: open-source, usado em servidores e programação
  • macOS: da Apple, para computadores Mac
  • Android: para celulares, baseado em Linux

Sistema de arquivos

Sistema de arquivos organiza como guardar e acessar dados no disco. Exemplos: NTFS (Windows), ext4 (Linux).

Modo texto (CMD)

No prompt de comando, usamos comandos como 'dir' para listar arquivos, 'cd' para entrar em pastas, 'copy' para copiar.

Modo gráfico (Explorer)

No Windows Explorer, navegamos clicando em pastas, arrastando arquivos, com ícones e menus.

Software de escritório

Programas para trabalho administrativo e produtivo, como fazer documentos, planilhas e apresentações.

Exemplos:

  • Microsoft Word: cria e edita textos. Usado para cartas, relatórios, currículos.
  • Microsoft Excel: faz planilhas e gráficos. Para cálculos financeiros e análise de dados.
  • Microsoft PowerPoint: cria apresentações visuais. Para slides em reuniões e aulas.
