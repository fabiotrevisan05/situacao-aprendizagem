Lógica de Programação

1. O que são Linguagens de Programação?

Linguagens de programação são conjuntos de regras e símbolos usados para criar instruções que computadores entendem e executam.

C++

Linguagem compilada de alto desempenho, muito usada em sistemas, jogos e aplicações que exigem velocidade. Oferece controle baixo nível com ponteiros e memória.

JavaScript

Linguagem interpretada originalmente para web, funciona em navegadores e servidores (Node.js). Dinâmica, flexível e ideal para desenvolvimento web e aplicações interativas.

Java

Linguagem compilada orientada a objetos, usa máquina virtual (JVM) para portabilidade. Muito usada em aplicações empresariais, Android e sistemas de grande escala.

Python

Linguagem interpretada de sintaxe simples e legível, excelente para iniciantes. Muito usada em data science, inteligência artificial, automação e web.

PHP

Linguagem de script server-side usada principalmente em desenvolvimento web. Fácil de aprender e amplamente suportada em servidores de hospedagem.

2. Linguagens Interpretadas vs Compiladas

Linguagem Interpretada

O código-fonte é traduzido e executado linha por linha pelo interpretador em tempo de execução. Exemplos: Python, JavaScript, PHP. Vantagem: mais flexibilidade; Desvantagem: mais lenta.

Linguagem Compilada

O código-fonte é traduzido em código máquina antes da execução. Exemplos: C++, Java, Go. Vantagem: mais rápida; Desvantagem: menos flexibilidade.

3. O que é HTML e CSS?

HTML

HTML (HyperText Markup Language) é a linguagem de marcação que estrutura o conteúdo de páginas web. Define elementos como títulos, parágrafos, imagens e links.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar elementos HTML, definindo cores, fontes, layouts, espaçamento e posicionamento.

4. O que é uma IDE?

IDE (Integrated Development Environment) é um ambiente integrado que oferece editor de código, compilador/interpretador, depurador e outras ferramentas para desenvolvimento.

Exemplos

  • VSCode: Editor leve e poderoso com extensões (Microsoft)
  • IntelliJ IDEA: IDE robusta para Java e outras linguagens (JetBrains)
  • Visual Studio: IDE completa para .NET e C++ (Microsoft)
  • PyCharm: IDE especializada para Python (JetBrains)
  • Eclipse: IDE open-source para Java

6. Ordenação com Bubble Sort


// JavaScript - Bubble Sort

function bubbleSort(arr) {
    let n = arr.length;
    
    for (let i = 0; i < n; i++) {
        for (let j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // Troca
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}

let numeros = [19, 3, 5, 11, 7, 2, 18, 9, 15, 1, 14, 6, 20, 8, 12, 4, 17, 10, 16, 13];
console.log("Ordenado: ", bubbleSort(numeros));

7. Soma de 10 Números Aleatórios em Array


// JavaScript - Soma de array

function somaNumeros() {
    let numeros = [];
    let soma = 0;
    
    // Gera 10 números aleatórios de 1 a 220
    for (let i = 0; i < 10; i++) {
        let numero = Math.floor(Math.random() * 220) + 1;
        numeros.push(numero);
        soma += numero;
    }
    
    console.log("Números: ", numeros);
    console.log("Soma: ", soma);
    console.log("Média: ", soma / 10);
}

somaNumeros();

8. Matriz 3x3 de Números Aleatórios


// JavaScript - Matriz 3x3

function gerarMatriz() {
    let matriz = [];
    
    // Cria matriz 3x3
    for (let i = 0; i < 3; i++) {
        matriz[i] = [];
        for (let j = 0; j < 3; j++) {
            matriz[i][j] = Math.floor(Math.random() * 20) + 1;
        }
    }
    
    // Exibe matriz
    console.table(matriz);
    return matriz;
}

gerarMatriz();

9. Cálculo do IMC


// JavaScript - Cálculo do IMC

function calcularIMC(peso, altura) {
    let imc = peso / (altura * altura);
    
    let classificacao = '';
    if (imc < 18.5) {
        classificacao = 'Abaixo do peso';
    } else if (imc < 25) {
        classificacao = 'Peso normal';
    } else if (imc < 30) {
        classificacao = 'Sobrepeso';
    } else if (imc < 35) {
        classificacao = 'Obesidade grau 1';
    } else if (imc < 40) {
        classificacao = 'Obesidade grau 2';
    } else {
        classificacao = 'Obesidade grau 3';
    }
    
    console.log(`IMC: ${imc.toFixed(2)} - ${classificacao}`);
}

calcularIMC(75, 1.75); // Peso em kg, altura em metros

10. Conversão de Temperatura


// JavaScript - Conversão de temperatura

function converterTemperatura(valor, de) {
    let celsius, fahrenheit;
    
    if (de === 'C') {
        celsius = valor;
        fahrenheit = (celsius * 9/5) + 32;
    } else if (de === 'F') {
        fahrenheit = valor;
        celsius = (fahrenheit - 32) * 5/9;
    }
    
    console.log(`${valor}°${de} = ${celsius.toFixed(2)}°C = ${fahrenheit.toFixed(2)}°F`);
}

converterTemperatura(25, 'C');  // 25°C para °F
converterTemperatura(77, 'F');  // 77°F para °C

11. Busca Linear

A busca linear é um algoritmo simples que percorre sequencialmente os elementos de uma lista até encontrar o valor procurado ou chegar ao final. É eficiente para listas pequenas, mas lenta para listas grandes. O tempo de execução no pior caso é O(n), onde n é o número de elementos.


// JavaScript - Busca Linear

function buscaLinear(arr, alvo) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === alvo) {
            return i; // Encontrou, retorna índice
        }
    }
    return -1; // Não encontrou
}

let numeros = [5, 12, 3, 19, 7, 15, 8];
console.log("Posição do 19: ", buscaLinear(numeros, 19)); // Saída: 3
console.log("Posição do 50: ", buscaLinear(numeros, 50)); // Saída: -1
