Componentes Eletrônicos

Resistor

Limita a corrente elétrica no circuito. Serve para proteger componentes e controlar a intensidade da corrente.

Diodo LED

Converte energia elétrica em luz. Usado em iluminação, displays e indicadores luminosos.

Transistor

Controla grandes correntes usando uma corrente menor.

Circuito Integrado

Componentes muito pequenos em um chip pronto para usar.

Sensores em Sistemas Automatizados

  • Sensor de Temperatura

    • Mede temperatura do ambiente ou objetos
    • Usado em: ar condicionado, estufas, controle industrial

  • Sensor de Umidade

    • Detecta umidade no ar ou no solo
    • Usado em: irrigação automática, controle de clima

  • Sensor de Luminosidade

    • Mede intensidade de luz
    • Usado em: iluminação automática, painéis solares

Arduino

O que é?

Plataforma open-source para criar projetos eletrônicos. Combina hardware e software fáceis de usar.

Para que serve?

  • Automação residencial
  • Robótica
  • Internet das Coisas (IoT)
  • Projetos educacionais
  • Protótipos rápidos

Vantagens

  • Barato
  • Fácil de aprender
  • Open-source
  • Comunidade grande
  • Multiplataforma

Desvantagens

  • Processamento limitado
  • Pouca memória
  • Não serve para projetos industriais grandes

Como o Arduino Funciona?

O Arduino tem um microcontrolador que roda um programa enviado pelo computador. Ele lê sensores, processa dados e controla saídas como LEDs e motores.

Eletrônica Embarcada

É quando você coloca um sistema eletrônico dentro de um equipamento para ele executar uma função específica. O Arduino serve pra criar protótipos desse tipo de sistema.

Linguagem de Programação

O Arduino usa uma linguagem baseada em C/C++, simplificada para iniciantes. A IDE tem bibliotecas prontas que facilitam muito.

Entradas e Saídas no Arduino

Digitais

Trabalham com dois estados: ligado (HIGH - 5V) e desligado (LOW - 0V).

  • Ler botões
  • Ligar/desligar LEDs
  • Comunicação (I2C, SPI, UART)
  • Controlar relés

Analógicas

Trabalham com valores variáveis de 0 a 5V.

  • Ler sensores (temperatura, luz)
  • Controlar brilho de LEDs (PWM)
  • Controlar velocidade de motores
  • Ler potenciômetros

Lei de Ohm

V = R × I (Tensão = Resistência × Corrente)

Potência

Pode ser calculada de 3 formas diferentes

Código Arduino

Sensor analógico acionando LEDs

// Lê sensor e acende LED conforme o valor

const int sensorPin = A0;
const int ledVerde = 2;
const int ledAmarelo = 3;
const int ledVermelho = 4;

void setup() {
  pinMode(ledVerde, OUTPUT);
  pinMode(ledAmarelo, OUTPUT);
  pinMode(ledVermelho, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int valor = analogRead(sensorPin);
  float tensao = valor * (5.0 / 1023.0);
  
  Serial.print("Sensor: ");
  Serial.print(valor);
  Serial.print(" | Tensão: ");
  Serial.println(tensao);
  
  if (valor < 341) {
    // 0-33% - LED verde
    digitalWrite(ledVerde, HIGH);
    digitalWrite(ledAmarelo, LOW);
    digitalWrite(ledVermelho, LOW);
  } 
  else if (valor < 682) {
    // 34-66% - LED amarelo
    digitalWrite(ledVerde, LOW);
    digitalWrite(ledAmarelo, HIGH);
    digitalWrite(ledVermelho, LOW);
  } 
  else {
    // 67-100% - LED vermelho
    digitalWrite(ledVerde, LOW);
    digitalWrite(ledAmarelo, LOW);
    digitalWrite(ledVermelho, HIGH);
  }
  
  delay(500);
}

Diodos e Displays

Diodo

Deixa a corrente passar só em uma direção. Funciona como uma passagem que bloqueia corrente no sentido contrário. Serve para proteger circuitos.

Diodo LED

É um diodo especial que emite luz quando passa corrente. Mais econômico que lâmpadas comuns, dura muito e tem várias cores. Usado em tudo: iluminação, telas, sinalizadores.

Display de 7 Segmentos

São 7 LEDs organizados em forma de "8". Cada LED pode acender sozinho para formar números de 0 a 9 e algumas letras.
